◆ CountdownTimer()

virtual void IMinesweeperGame::CountdownTimer ( )
pure virtual

Method to the timer in a separate thread, continuously updating the remaining time. Notifies registered listeners about timer changes. If the timer reaches zero or the game state changes to a terminal state (not WIN or FIRSTCLICK), the game ends.

Implemented in MinesweeperGame, private method.

◆ EndGame()

virtual void IMinesweeperGame::EndGame ( )
pure virtual

Method to end the game:

  • Sets the game state to GAMEOVER if the game is over
  • Sets the game state to WIN if the game is won
  • Loops through the listeners and calls the OnGameOver or OnWin method
